# Dream Loop UI

Frontend UI project for the Dream Loop application built with React, Vite, and TypeScript.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- DaisyUI
- Framer Motion
- React Router DOM

---

## Features

- Modern responsive UI
- Animated page transitions using Framer Motion
- Component-based architecture
- Routing with React Router
- Tailwind CSS styling
- DaisyUI components
- Fast development environment with Vite
